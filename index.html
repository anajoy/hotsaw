Hotsaw

Hotsaw contains some helpful open source file shiper configuration files and scripts for handling the difficult task forwarding dynamic WebSphere system and access logs to logstash and elastic. It has been build on top log-courier and zeromq.

Authors and Contributors

In 2012, Jordan Sissel (@jordansissel) developed a lightway log-shiper named lumberjack, runned afterwards as logstash-forwarder. Jason Woods (@driskell) improved 2014 the functionalities significantly adding some nice features and eliminating major bugs. His branch is remaned log-courier and still maintained. The existing logstash-forwarder has been meanwhile incorportated in filebeat and further developed by the community of elastic.
